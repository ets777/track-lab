# About TrackLab

## Main Purpose

The main purpose of the app is to let user track their activities. Tracking implies inputing data about what user have done, metrics values, tags and custom items.

## Sections

### History

History section stores user activities grouped by days.

### Lab

Lab contains elements, such as metrics and custom lists, that user can customize to make tracking more informative.

### Stats

Stats display information from user activities in graphs to let user analyze their behaviour. Graphs in statistics section displays dynamics of metrics changes through time and information about time spent on specific actions or tags or custom list items.

### Settings

Settings section let user change UI language and work with database.
